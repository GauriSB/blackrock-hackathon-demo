# BlackRock Self-Saving API

## Setup
1. Build: `./mvnw clean package`
2. Docker: `docker build -t blk-hacking-ind-gauri-barge .`
3. Run: `docker run -p 5477:5477 blk-hacking-ind-gauri-barge `

## Features
- **Micro-investing**: Automated rounding of expenses.
- **Temporal Logic**: Complex $q$ and $p$ period rules with priority handling.
- **Performance**: Real-time memory and thread monitoring.
